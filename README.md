# attentional-blink

更新时间：2026-04-27

本工作区围绕两条相互关联的研究主线展开。第一条主线关注内部注意与外部注意的交互作用，具体问题是：当被试将注意定向到胃部等内部身体信号时，外部视觉时间注意是否会受到影响，以及这种影响能否通过 attentional blink 范式加以测量。第二条主线关注特殊人群，尤其是厌食症及相关进食病理群体，对食物刺激的加工方式是否发生改变；更具体地说，食物在这些群体中究竟仍然表现为奖赏性高显著刺激，还是被转化为威胁性、回避性或冲突性刺激。

本目录中的文档按照功能分工。`01_研究问题梳理_内感受_注意_AB.md` 用于说明整体研究逻辑，是理解课题结构、研究问题与理论假设的主入口。`02_阅读顺序与流程.md` 用于安排文献阅读路径，帮助将阅读工作转化为逐步推进的研究过程。`03_工作说明与问题记录.md` 保存项目配置、版本变动和工作记录。`04_内感受_外感受_定义与文献梳理.md` 负责厘清内感受、外感受及其关系的概念基础。`05_食物刺激_厌食_AB相关研究梳理.md` 用于梳理进食障碍、食物显著性与 attentional blink 的相关研究。`06_Table1_Clinical_AB_文献中文摘要.md` 汇总临床人群中 attentional blink 研究的代表性文献。`07_Attentional_Blink_最终整合版.md` 提供一份较完整的 AB 理论与发展综述。`08_相关问题问答记录.md` 用于持续记录与本课题相关的概念、方法与设计问题。

如果是第一次进入该项目，建议的阅读顺序是：先阅读 `01`，把研究问题和两条主线的关系建立起来；再阅读 `02`，明确文献和研究推进的优先级；如果需要了解项目背景、配置、历史修改或合作过程，再阅读 `03`。在此基础上，再根据具体问题进入 `04` 至 `08` 的专题文档。整个工作区默认服务于同一目标，即围绕内部身体信号、外部视觉时间注意以及特殊人群中的食物加工偏向，逐步搭建一个可以实证检验的 attentional blink 研究方案。
